# Installation

## Prerequisites

- Python 3.8+
- PostgreSQL (for data storage)

## Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/youruser/pyppetdb.git
   cd pyppetdb
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   python -m pyppetdb.main
   ```
