# Introduction

Welcome to the pyppetdb documentation.

pyppetdb is a PuppetDB-compatible API server and Hiera backend, designed to provide a flexible and scalable way to manage your Puppet infrastructure.

## Key Features

- **PuppetDB Compatibility**: Implements the PuppetDB API for seamless integration with Puppet.
- **Hiera Backend**: Provides a powerful Hiera backend for managing node data.
- **CA Management**: Integrated Certificate Authority management.
- **Job Execution**: Built-in job execution system for managing remote nodes.
